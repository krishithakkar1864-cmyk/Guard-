# Guard Project

A Django-based guard management application.

## Features
- Guard management and booking flows
- Service registration and profile handling
- Django templates, static assets, and database-backed forms

## Setup
1. Create and activate a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Apply migrations:
   ```bash
   python manage.py migrate
   ```
4. Run the development server:
   ```bash
   python manage.py runserver
   ```

## Project structure
- `Guard Project/` contains the Django application code and templates.
- `static/` holds site assets.
- `templates/` contains HTML templates.
